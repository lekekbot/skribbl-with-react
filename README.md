# Installation:

### Starting the Servers :

1. To start backend server:
```   
$ cd backend
$ npm i
$ npm start
```
2. to start frontend server:
```
$ cd frontend
$ npm i
$ npm start
```

Wait for everything to load. 
<br><br>
### To access the app, go to:<br>
```
http://localhost:3000/
```
